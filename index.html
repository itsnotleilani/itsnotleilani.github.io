Network flow - find the cut value

Welcome! This website is designed to help people learn how to find the cut value of a directed graph.

But first of all, what is a cut value? A cut value is when you take a graph and "cut" it into 2 groups. In an s-t cut, one group must contain s, and the other must contain t. The cut value is the number of edges that go from the s group into the t group. An edge coming from the t group to the s group does not contribute to the cut value.

It may help to imagine the node groups as bodies of land, and the s-t cut as a body of water separating the two groups, hence the dolphin :)

For instance, in this graph, { s, a, b } is in the s group, and only { t } is in the t group. Then we look at the edges that connect the groups, and see that there is only 1 edges. This makes our cut value = 4.

On this website, you select nodes to add to the s group (brown). Any unselected nodes will be placed in the t group (green). The edges connecting the two groups will turn blue, helping you to figure out the cut value.

You can use the graph to figure out the cut value, and then the input field to check your answer.
